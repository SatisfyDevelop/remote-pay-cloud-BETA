This is an example of the Clover High Level Cloud API. It relies on the device to be running the Cloud Pay Display app. It uses the device id to identify the device, and a authorization token that was previously obtained. It makes a connection to a device, and charges $123.45.
NOT: Printed tips do not reflect the custom tip amount at this time (CLOVER-18001) Once the authorization is complete, an adjustment is made for the tip. The result of the transaction is displayed below.
